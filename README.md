# inter-DEX-arb-screener

This is an inter-DEX arb screener dapp within Uniswap V3, and Quickswap V3 DEXs in Polygon PoS chain. 

Link: https://tinyurl.com/inter-dex-arb-screener

In the Dapp always click to enter "Route Specification" link to verify pricing difference of each arbitrage signal token in both buy DEXs, and sell DEXs. The pricing difference must be present.
