# inter-DEX-arb-screener

This is an inter-DEX arb screener site within Uniswap V3, and Quickswap V3 DEXs in Polygon PoS chain. 

Link: https://tinyurl.com/inter-dex-arb-screener

In the site always click to enter "Route Specification" link to verify pricing difference of each arbitrage signal pair present in both buy DEXs, and sell DEXs in this site with Dexscreener.com site with using given contract address in signal to check if pricing difference also be present there.

You can use Morpho Blue flash loan to add liquidity something like 80M WPOL/USDC/USDT in Polygon PoS chain in the sell DEX token pair liquidity pool contract address with a range like:-0.01% lower to -0.02% lower from market price of token pair in sell DEX. Then price impact of base token to quote token swap in sell DEX can be reduced - if you start with upto 10000 WPOL/USDC/USDT notional value in buy DEX using an inter-DEX arbitrage signal given by my arbitrage screener site.
