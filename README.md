# inter-DEX-arb-screener

This is an inter-DEX arb screener site within Uniswap V3, and Quickswap V3 DEXs in Polygon PoS chain. 

Link: https://tinyurl.com/inter-dex-arb-screener

In the site always click to enter "Route Specification" link to verify pricing difference of each arbitrage signal pair present in both buy DEXs, and sell DEXs in the site with Dexscreener.com site with using given contract addresses in signals to check if pricing differences also be present there.
